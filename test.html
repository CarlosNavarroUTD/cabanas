El desarrollo del proyecto fue un desafío de aprendizaje continuo, centrado en construir una API robusta con Django Rest Framework, lo que requirió dominar conceptos avanzados como serialización, vistas, relaciones complejas entre modelos, permisos, optimización de consultas y autenticación con tokens JWT, junto con estrategias para proteger los datos. Fue un viaje de aprendizaje continuo, especialmente en la construcción de una API robusta y eficiente mediante Django Rest Framework, lo que representó un desafío inicial significativo al profundizar en conceptos avanzados como serialización, creación de vistas y manejo de endpoints RESTful. Configurar una API capaz de gestionar múltiples modelos de datos (cabañas, actividades, usuarios y reservas) implicó dominar relaciones complejas entre modelos, gestión de permisos y autorizaciones, y la optimización de consultas, convirtiéndose en áreas clave de expertise. La autenticación con tokens JWT presentó retos adicionales, requiriendo un entendimiento profundo de los mecanismos de seguridad web y la implementación de estrategias para proteger los datos.
